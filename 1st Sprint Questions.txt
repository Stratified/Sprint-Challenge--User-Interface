1)  If you were to describe semantic HTML to the next cohort of students, what would you say?
Specific HTML element tags that you should prioritize. They're easier for the computer to read and bolster your SEO.
2)  What are the 4 areas of the box model?
Content, padding, border, margin.
3)  While using flexbox, what axis does the following property work on: align-items: center?
The vertical/Y axis.
4)  Explain why git is valuable to a team of developers.
It allows multiple developers to be able to work on one project at the same time and it includes failsafes and backups
to your content. 
5)  Define mobile-first design in your own words.
Designing a website starting with the mobile layout first and expanding to a desktop design.
